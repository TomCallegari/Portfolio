Temperatures

Intuitively, we can say that as you get closer to the equator, the temperature will rise. But how can we be certain? Or atleast come close to some approximation of certainty? This static dasboard website aims to visualize an answer to this question using the Python plotting library Matplotlib. These simple scatter plot diagrams show varying degrees of relationship between Latitude and city Temperatures, Humidity levels, Cloudiness and Wind Speeds.

Data for this project was gathered through the OpenWeatherMap API. A script was run collecting 3500+ randomly selected cities real-time temperature data over the course of 5 hours. It was then cleeaned and visualized to give the reader a clearer understanding of the relationship between a selected cities Latitude and and it's weather.
