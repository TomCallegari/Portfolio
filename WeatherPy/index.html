Temperatures

Intuitively, we can say that as you get closer to the equator, the temperature will rise. But how can you be absolutely certain? Or atleast as close to certain as possible? This project, a static dashboard website, aims to give some level of certainty to this question.

Through the use of the OpenWeatherMap API 3500+ randomly collected cities real-time temperature data was collected, cleaned and visualized to give the reader a clearer understanding of the relationship between a cities latitude and and it's weather.
